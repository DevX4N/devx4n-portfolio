## Como executar o projeto

### Requisitos

- Node.js 22.13 ou superior
- pnpm

### Passos

```bash
git clone https://github.com/DevX4N/devx4n-portfolio.git
cd devx4n-portfolio
pnpm install
pnpm dev
```

Depois, abra o endereço exibido no terminal para visualizar o site.

Para gerar a versão de produção:

```bash
pnpm build
```

## Estrutura principal

- `app/`: páginas, componentes e estilos do portfólio
- `public/`: imagens e arquivos públicos
- `worker/`: integração com o ambiente Cloudflare
- `.openai/hosting.json`: configuração de hospedagem do Sites
